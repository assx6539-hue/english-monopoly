# 🎲 英文大富翁 PWA

高中英文教學版大富翁遊戲。題庫採用康軒高中英文 L8 《Too Much of a Good Thing — Overtourism in Barcelona》課文內容。

## 🎮 玩法

- 2–6 人輪流擲骰子前進
- 踩到題目格抽題答題（造句 / 句型轉換 / 翻譯 / 單字選擇）
- 答對 / 答錯都換下一位玩家
- 答錯只停留原地（不退格、不扣分）
- 只有抽到機會 / 命運卡才可能退格
- **最先繞完一圈回到起點者獲勝 🏆**

## ✨ 功能

- 🐱🐶🐰🐼🦊🐨 動物頭像
- 🔊 Web Audio 合成音效 + 背景配樂
- 📐 全螢幕模式
- ✏️ **老師專區**：可線上編輯 / 新增 / 匯入 / 匯出題庫
- 📱 PWA 支援：可安裝為手機 App、離線可玩

## 📚 課文範圍

- **單字** 18 個（overflow, narrow, extent, decline, dramatically, flock, expansion, invasion, complaint, thoughtless, tenant, landlord, partial, measure, thorough, strive, negative, overtourism）
- **句型** Only + 倒裝句、N times as ... as 倍數比較
- **慣用語** be fed up with, to such an extent that, put on the map, double-edged sword
